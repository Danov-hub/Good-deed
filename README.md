# Good-deed
ДОБРОЕ ДЕЛО
Возможности:

- Чтение списка фамилий учеников из текстового файла students.txt;

- Генерация случайных чисел от -10 до 10 для определения направления и количества шагов;

- Вывод подробного протокола игры в реальном времени;

- Автоматическое начисление рейтинга ученикам, совершившим «доброе дело»;

- Формирование итогового отчёта с сортировкой учеников по убыванию рейтинга;

- Сохранение исходного порядка учеников при равных рейтингах;

- Защита от некорректного пользовательского ввода;

- Циклическое меню с возможностью начать новую игру или выйти из программы.


Режимы работы:

- Начать игру (с указанием количества раундов);

- Выход из программы.
